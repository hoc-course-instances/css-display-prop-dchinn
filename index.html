Block

Block-level elements always start on a new line and take up the full width available (stretching left to right as far as possible)

BLOCK
BLOCK
Examples include div, nav, header, section, footer, and ul elements.

Inline

Inline elements do not start on a new line and only occupy the space their content requires. INLINE Some example of inline elements include <span>, <input> ,<label>, <select> and <a> elememts.

Inline-block

Inline-block is set by the CSS display property and differs to inline elements as we can set width and height. We cannot do this width inline elemetns as they only take the space their content needs. Also, top and bottom margins and paddings are respected, but inline elements they are not. INLINE-BLOCK . Unlike block-level elements, inline-block elements do not start on a new line and will sit next to other elements.
